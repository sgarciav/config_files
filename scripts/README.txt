Run the get and set scripts to update the config files in the repo and set
them up in the current machine, repsectively.

Usage:
	$ source set_config <directory or filename>
OR
	$ source get_config <directory or filename>

WARNING: The 'set_config.sh' script will delete local files and replace them
with the ones from the repo. There are no security checks in place, so be
completely sure you want to run.
